# GO + HTMX

- go webserwer
- html templates
- htmx for interactivity
- tailwind css for styling

## Commands

- `make dev` - start dev server on port 3000
- `make tw-watch` - generate tailwind styles for dev

### Prerequisites

To run make commands, you need to have:

- gin
- tailwindcss cli
